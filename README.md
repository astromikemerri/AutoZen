# AutoZen
A new skin to repurpose the microcar project as an automated Zen Garden that rakes itself.

To use this project, you need to have built the <A href=https://github.com/astromikemerri/Microcar>Microcar Project</a> which provides the underlying hardware of linear motors that drive a small magnetic puck.  This repository provides <A href=autozen.3mf>a new skin that is the Zen garden</a>, including accessories of a sand smoother and stylus-remover, and <A href=AutoZen_Distribution.ino>a new sketch</a> that creates the patterns in the sand.

You will also need to source a couple of <A href=https://www.first4magnets.com/product/5mm-dia-x-2mm-thick-n35-neodymium-magnet-051kg-pull-19444>5mm-diameter x 2mm deep cylindrical magnets</a> for the picker-upper, and an <A href=https://www.magnosphere.co.uk/sphere-magnet-3mm-dia-n40-nickel-neodymium-ndfeb-rare-earth-magnetic-holds-130-g/a-1903/>3mm diameter spherical magnet</a> for the under-sand stylus.  In addition, you will need some <A href=https://www.amazon.co.uk/Craft-Zen-Garden-Aquarium-Terrarium/dp/B0CBMK25B2/>suitable sand.  

Print the garden and accessories, glue two cylindrical magnets into the stylus remover (oriented to be attracted toward the puck in the base) install the garden on the base (you may have to file out the locating holes a little, depending on how well your printer behaves), fill the pit with sand, upload the sketch, and you will have one of these:
